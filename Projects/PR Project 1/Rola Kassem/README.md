# 📢 Public Relations (PR) Project

## 📌 Project Overview
This project is a Public Relations (PR) project designed to promote an online learning platform (Udemy). The goal of the project is to apply PR strategies to advertise and improve awareness of the platform and its services.

---

## 🎯 Objective
The main objective of this project is to create an effective promotional strategy for Udemy, focusing on increasing visibility, engagement, and user interest through structured PR techniques.

---

## 📚 What We Learned in PR
Through this project, we learned:

- Fundamentals of Public Relations and its importance in modern communication
- How to build and manage a PR campaign
- Audience analysis and targeting strategies
- Media and digital communication methods
- Brand positioning and reputation management
- How advertising impacts user perception and engagement

---

## 📊 Project Focus
- Platform: Udemy

## 📊 MP (Marketing Plan)

The CMP (Marketing Plan) in this project refers to the structured strategy used to promote Udemy effectively through targeted advertising and communication methods.

It helped us define how the advertisement would be executed, including planning, audience targeting, and promotional techniques.

The MP was used to:
- Identify the target audience for Udemy
- Plan the advertising strategy and key messages
- Choose suitable digital platforms for promotion
- Structure the campaign in a clear and organized way
