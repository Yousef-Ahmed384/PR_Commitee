# 🌍 About OSC

## ✨ What is OSC?

**OSC (Open Source Community)** is an open, collaborative group at FCIS (Faculity of Computer and Information Systems) ASU (Ain Shams Universty) focused on promoting open-source culture, tools, and knowledge.

OSC contains multiple specialized committee — the **PR Committee** is one of them.

# 🎤 What is PR?

## 🌟 Public Relations

Public Relations is not only about speaking or presentations.

PR is the ability to:
- communicate effectively
- influence professionally
- represent ideas clearly
- understand people
- build relationships
- solve communication problems
- lead discussions confidently
- create impact through communication

It combines:
- communication
- leadership
- emotional intelligence
- teamwork
- branding
- strategy
- public speaking
- organization
